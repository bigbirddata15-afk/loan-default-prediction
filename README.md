# Loan Default Prediction System

A comprehensive microfinance web application built with Streamlit for loan default risk assessment and digital banking operations.

## Features
- User authentication and registration
- Loan default risk prediction using ML
- Account management (deposits, withdrawals, transfers)
- Mobile recharges
- Financial forecasting
- Transaction history
- MongoDB integration

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
